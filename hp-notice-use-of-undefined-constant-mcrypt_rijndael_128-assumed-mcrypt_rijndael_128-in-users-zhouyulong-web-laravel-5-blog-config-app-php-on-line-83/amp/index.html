竟然有报错信息，真的好晕了，怎么办？

mcrypt是php里面重要的加密支持扩展库,需要安装这个吧！

需要安装mcrypt

$ brew install mcrypt

$ brew install php56-mcrypt
==> Installing php56-mcrypt from homebrew/php
Error: You must `brew link jpeg unixodbc` before homebrew/php/php56-mcrypt can be installed

有报错了，安装php56-mcrypt之前要命令运行如下：

$ brew link jpeg unixodbc
Linking /usr/local/Cellar/jpeg/8d... 0 symlinks created
Linking /usr/local/Cellar/unixodbc/2.3.4... 0 symlinks created

好了，直接可以命令运行：

$ brew install php56-mcrypt
