Mac下通过Homebrew安装MySQL

2016-10-21   |   IT Mac

学习cakephp的时候，不能连接mysql服务器的报错，好像是Mac没有自带的mysql，我就只好通过homebrew安装mysql。

我觉得brew安装什么东西真的方便了，如果没有brew的话，建议你去http://brew.sh/index_zh-cn.html

安装mysql：

 brew install mysql

安装的时候下载需要时间，这个不用着急吧。你也可以利用等待的时间去喝咖啡或者消磨一些你想要做的事情吧。

Mac2终端器安装成功mysql服务器

==> Downloading https://homebrew.bintray.com/bottles/mysql-5.7.13.el_capitan.bot
######################################################################## 100.0%
==> Pouring mysql-5.7.13.el_capitan.bottle.tar.gz
==> /usr/local/Cellar/mysql/5.7.13/bin/mysqld --initialize-insecure --user=zhouy
==> Caveats
We've installed your MySQL database without a root password. To secure it run:
    mysql_secure_installation

To connect run:
    mysql -uroot

To have launchd start mysql now and restart at login:
  brew services start mysql
Or, if you don't want/need a background service you can just run:
  mysql.server start
==> Summary

设置 MySQL 用户以及数据存放地址:

$ unset TMPDIR
$ mysql_install_db --verbose --user=`whoami` --basedir="$(brew --prefix mysql)" --datadir=/usr/local/var/mysql --tmpdir=/tmp

启动mysql服务:

$ mysql.server start
