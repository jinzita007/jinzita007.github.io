Hexo换主题的方法

2016-10-02   |   Hexo

我觉得Hexo的主题好看，因为简单而且干净的主题，所以我喜欢Hexo的优点。接下来我就告诉你怎么用好的主题模版，可以按照我的方法就去做。
进入blog的目录

cd blog

安装主题模版

npm install --save hexo-renderer-jade hexo-generator-feed hexo-generator-sitemap hexo-browsersync hexo-generator-archive

下载主题模版

clone https://github.com/pinggod/hexo-theme-apollo.git themes/apollo

生成项目

hexo generate

部署到Github

hexo deploy

简写命令

hexo n "我的博客" == hexo new "我的博客" #新建文章
hexo p == hexo publish
hexo g == hexo generate#生成
hexo s == hexo server #启动服务预览
hexo d == hexo deploy#部署
