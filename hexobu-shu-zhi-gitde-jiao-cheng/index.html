Hexo部署至Git的教程

2016-10-02   |   Hexo

如何Hexo项目发布到Git？

GIt

为了支持github，需要安装另一个插件
进入blog的目录：

cd blog

安装hexo-deployer-git

npm install hexo-deployer-git --save

然后生成web项目

hexo g

启动服务器预览

hexo s -o

修改_config.yml文件

这是本地项目，现在想要发布到github的话，需要修改_config.yml文件，打开它找到最后一行，我们把github的配置信息填上去。

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
    type: git
    repo: https://github.com/myweb/myweb.github.io.git
    branch: master

上传操作

hexo d

在浏览器输入myweb.github.io可以看到效果

请看一下《Hexo安装和配置》文章，然后如何把Hexo部署到Git。
