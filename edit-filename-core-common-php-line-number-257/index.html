2016-11-10   |   php

在PHP分配表达式中，总是返回赋值。 所以$ _config [0] =＆$ config返回$ config - 但不是变量本身，而是它的值的副本。 返回一个临时值的引用将不是特别有用（改变它不会做任何事情）。

在Common.php找到257行:

return $_config[0] =& $config;

修改之后：

$_config[0] =& $config;
return $_config[0];
