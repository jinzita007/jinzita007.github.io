编写简单的计算标准体重的程序

2016-11-02   |   C

编写一段程序，如果你想知道自己的体重是否达到标准体重，就可以请输入你的身高就能计算标准体重的实数值。

建立a_03.c文件：

#include <stdio.h>

int main(void) {
  int na;
  double ave;
  printf("请输入您的身高:");
  scanf("%d", &na);
  ave = (double)(na - 100) * 0.9;
  printf("标准体重是%4.1f\n", ave);
  
  return 0;
}

编译文件：

localhost:c jinghua$ make a_03
cc     a_03.c   -o a_03

打开程序：

localhost:c jinghua$ ./a_03
请输入您的身高:180
标准体重是72.0
