PHP框架篇（收集）

2016-11-16   |   php

我先学好前端的基础，然后有时间再学习PHP，主要是要掌握PHP原生，如果不会写原生，就不要去学习其他mvc框架。我认为一边学习PHP原生以及OOP思想，一边学习所有的MVC框架，但是要精通一个MVC框架就足够了。

CodeIgniter

CodeIgniter 是一个小巧但功能强大的 PHP 框架，作为一个简单而“优雅”的工具包，它可以为开发者们建立功能完善的 Web 应用程序。

CodeIgniter中文官网：http://codeigniter.org.cn/

Laravel

Laravel是一套简洁、优雅的PHP Web开发框架(PHP Web Framework)。它可以让你从面条一样杂乱的代码中解脱出来；它可以帮你构建一个完美的网络APP，而且每行代码都可以简洁、富于表达力。

Laravel中文官网：http://www.golaravel.com/

ThinkPHP

ThinkPHP是为了简化企业级应用开发和敏捷WEB应用开发而诞生的。

ThinkPHP中文官网：http://www.thinkphp.cn/

ThinkCMF

ThinkCMF是一款基于PHP+MYSQL开发的中文内容管理框架。ThinkCMF提出灵活的应用机制,框架自身提供基础的管理功能,而开发者可以根据自身的需求以应用的形式进行扩展。

ThinkCMF中文官网：http://www.thinkcmf.com/
