Linux安装免费SSL证书

2018-01-28   |   Nginx SSL HTTPS

Let's Encrypt是国外一个公共的免费SSL项目，由 Linux 基金会托管。这就是我找到了免费SSL证书。然后我一步一步地安装SSL证书，再启动nginx服务，就打开没问题了。

下载letsencrypt文件

# git clone https://github.com/letsencrypt/letsencrypt

打开letsencrypt文件

# cd letsencrypt

运行letsencrypt-auto

# ./letsencrypt-auto
