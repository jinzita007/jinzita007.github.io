安装symfony的报错

2016-10-15   |   php

创建Symfony程序

c:\projects> php symfony new my_project_name

安装symfony的时候报这个错误：

[GuzzleHttp\Exception\RequestException] cURL error 60: SSL certificate problem:......

最佳解决方法：curl本地验证问题， 需要修改php.ini 里面的curl 配置。

打开https://curl.haxx.se/ca/cacert.pem
然后保存一下，就放在php5文件夹下

打开php.ini里面加上这个：

curl.cainfo = "D:\phpStudy\php55\cacert.pem"

这样可以解决curl本地验证问题吧。
