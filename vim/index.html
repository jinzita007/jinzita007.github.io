vim编辑器设置行号及缩进

2016-11-21   |   vim

vim编辑器默认没有显示行号，也许可能会影响工作效率。如果是大师或者牛逼的人可以忽略我写的文章吧。vim编辑器要设置显示行号及缩进等，有了显示行号的话，就容易看的清楚了吧。

用vi或者vim可以命令文件，vi是没有颜色，vim是有颜色，要看你喜爱用哪个命令就行了。

vi /etc/vimrc

vimrc的文件配置

#显示行号
set nu
#设置缩进
set tabstop=2

最后保存一下
如果用搜狗输入法，肯定不能用命令保存一下吧。最好默认英文输入法，按键esc的时候，再输入:wq保存就0K.

:wq
