React Native环境搭建

2016-11-21   |   Javascript React

据说Facebook产品中使用了React Native，持续地投入建设React Native。React Native能提高多平台开发的开发效率，可以做成一流原生的app。我第一次开始学习React Native，也许是很好的学习体验和乐趣。接下来我们开始搭建基本的React Native环境，然后编写hello world的程序。

我目前现在使用mac系统,必须安装Homebrew。
Homebrew是Mac系统的包管理器，用于安装Nodejs和其他必需的工具。

安装Homebrew

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装Node

使用Homebrew来安装Node

brew install node

安装完成之后，查看是否成功安装Node和Npm的存在

node -v

npm -v

Yarn、React Native的命令行工具

npm install -g yarn react-native-cli

如果看到EACCES: permission denied这样权限的报错

sudo chown -R `whoami` /usr/local

使用淘宝npm镜像可以加速下载

npm config set disturl https://npm.taobao.org/dist --global

测试安装

react-native init AwesomeProject
cd AwesomeProject
react-native run-ios
