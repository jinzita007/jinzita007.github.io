SolrCore初始化失败:Could not load conf for core new_core

2018-03-25   |   solr

当我继续玩solr的时候，试图通过管理界面和命令行添加一个新的Core。

但在这种情况下，都会遇到以下错误：

Could not load conf for core new_core: Error loading solr config from /Users/zhouyulong/solr/server/solr/new_core/conf/solrconfig.xml

因为new_core没有配置好，已经解决如下

zhouyulongdeMacBook-Pro:solr zhouyulong$ bin/solr create -c ceshi
WARNING: Using _default configset. Data driven schema functionality is enabled by default, which is
         NOT RECOMMENDED for production use.

         To turn it off:
            curl http://localhost:8983/solr/ceshi/config -d '{"set-user-property": {"update.autoCreateFields":"false"}}'

Created new core 'ceshi'
zhouyulongdeMacBook-Pro:solr zhouyulong$
