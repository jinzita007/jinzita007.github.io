更改pip源使用国内镜像

2018-12-28   |   python pip

当pip安装lxml的时候太慢了，浪费时间。不过我们给pip源使用国内镜像，可以大幅提升下载速度。

1.创建一个.pip文件夹

~ mkdir .pip

2.进入.pip文件夹

~ cd .pip

3.创建一个pip.conf文件

~ touch pip.conf

4.vim打开pip.conf

~ vi pip.conf

5.写入配置

# 写入配置
[global]
index-url = http://mirrors.aliyun.com/pypi/simple/
[install]
trusted-host = mirrors.aliyun.com
