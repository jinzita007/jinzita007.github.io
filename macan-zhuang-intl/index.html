Mac安装intl

2016-09-24   |   Mac

Mac要安装intl，因为我的CakePHP不能缺少intl模块。这样我准备呀，如果有了intl扩展的增加，我可以终于学习CakePHP啦！

1.使用 Homebrew 安装 ICU 库

brew install icu4c

2.安装 intl

sudo pecl install intl

3.编辑 /etc/php.ini

在最后添加： extension=intl.so

4.重启 Apache

sudo apachectl restart

5.检查是否安装成功

php -m | grep intl #正常会返回 'intl'
