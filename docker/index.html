Docker容器报错

2019-04-28   |   Docker Linux

守护程序的错误响应：驱动程序在端点some-mysql外部连接失败

启动Docker容器报错信息如下：

/usr/bin/docker-current: Error response from daemon: driver failed programming external connectivity on endpoint some-mysql (afff9c2dab0bd38835b3b41d14f002ff4606b62b1046d9fb23646a5c47ac582d):  (iptables failed: iptables --wait -t nat -A DOCKER -p tcp -d 0/0 --dport 3306 -j DNAT --to-destination 172.17.0.2:3306 ! -i docker0: iptables: No chain/target/match by that name.
 (exit status 1)).

解决方式：重启启动

systemctl restart docker
