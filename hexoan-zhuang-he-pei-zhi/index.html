Hexo安装和配置

2016-10-02   |  

Hexo是一个简单地、轻量地、基于Node的一个静态博客框架，可以方便的生成静态网页托管在github和Heroku上。

安装

1.Hexo是基于Node,必须要安装Node和npm。你可以自行去安装Node和npm！

2.如果安装成功了的话，就开始安装Hexo吧。

npm install hexo-cli -g  #-g表示全局安装, npm默认为当前项目安装

注意：如果是Mac操作系统安装Hexo，按照上面的命令去安装时候可能会报错，需要加上sudo就不会有报错

sudo npm install hexo-cli -g

3.执行init命令初始化到你指定的blog目录

hexo init blog

4.进入blog目录

cd blog

5.npm安装

npm isntall #install before start blogging

6.运行本地服务

hexo server

7.Mac终端运行本地服务:

bogon:blog zhu$ hexo server
INFO  Start processing
INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

8.浏览器输入http://localhost:4000就可以看到效果。

目录结构

.
├── .deploy       #需要部署的文件
├── node_modules  #Hexo插件
├── public        #生成的静态网页文件
├── scaffolds     #模板
├── source        #博客正文和其他源文件, 404 favicon CNAME 等都应该放在这里
|   ├── _drafts   #草稿
|   └── _posts    #文章
├── themes        #主题
├── _config.yml   #全局配置文件
└── package.json
